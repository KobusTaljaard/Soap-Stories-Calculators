# Steffi’s Soap Stories – Calculators

This app is the "Master Batch Calculator" for your luxury soap brand.
It is built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

For setup and deployment, see below:

## Quick Start

- You do **not** need to code to use or deploy this app!

- Add your Supabase keys in Vercel as needed.
